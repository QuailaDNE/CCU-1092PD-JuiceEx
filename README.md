# CCU-1092PD-JuiceEx
The 3 programs are the exercises from Juice (the online judge system of CCU.)

## 18_Copy.c
This program will copy the contents from one file to another.
### command: 
``` terminal
./a.out path1  path2
```
### description:
The contents of file in path1 will be copied to the file in path2.
